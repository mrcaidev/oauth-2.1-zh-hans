# 序言

## 摘要

OAuth 2.1 授权框架让一款应用能够获取对受保护资源的有限访问权限。这款应用既可以代表资源所有者，协调资源所有者和授权服务之间的批准交互；也可以代表它自己，允许应用获取访问权限。本规范取缔并废弃了 RFC 6749 所述的 OAuth 2.0 授权框架，以及 RFC 6750 所述的不记名令牌用法。
